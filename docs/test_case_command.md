

## chat bot 聊天测试

## 简单任务
**image 图片处理**
```chat simple image

  两只大黄狗在打架，架得特别凶，嘴是毛，互不相让 写实风格，
```
**邮件处理**

```chat simple mail

  两只大黄狗在打架，架得特别凶，嘴是毛，互不相让 写实风格，
```

**pdf图片提取**

```chat pdf image extract 

  帮我提取xxxx.pdf中的图片 
  - 保持图片的完整性，不要将一张图片切分为多个部分
  - 提取完整的原始图片，而不是按页面区域切分
  - 保持图片的原始尺寸和格式
```
